# go-caniuse

Simple command line tool for [caniuse](https://caniuse.com) written by Golang.    


# Usage

$ go-caniuse websockets


$ go-caniuse -h

Use like this: 

```
$ caniuse websocket
```

You can specify output format, like xml or json, and yaml.
```
$ caniuse websocket -o json
```
