# Abstract

# Installation

## MacOS

You can use homebrew on Mac OS.

```
$ brew tap stqp/go-caniuse/go-caniuse
$ brew install go-caniuse
```

# Usage

$ caniuse -h
$ caniuse [Web Technology]

Use like this:

```
$ caniuse websocket
```

You can specify output format, like xml or json, and yaml.
```
$ caniuse websocket -o json
```
